# HR Attrition Predictor

Streamlit-дашборд для прогнозирования риска увольнения сотрудников на основе HR-данных и данных учёта посещаемости (СКУД). Модель — XGBoost, с объяснением решений через SHAP.

## Возможности

- Обзор HR-показателей компании
- Анализ риска увольнения по конкретному сотруднику
- Пакетный анализ по загруженному CSV
- Метрики качества модели
